
# Al-Afiuya Rehabilitation Center

Core Features:

- Secure user authentication with login/registration
- Complete patient registration and profile management
- Daily treatment tracking and history
- Payment status monitoring with automated notifications
- Dashboard with key performance indicators and statistics
- Patient discharge management system
- Notification center for pending payments
- Responsive design for desktop and mobile use
